# NewsNation

News Nation is an android based news fetching app.

app that contains the following:

1.) Written in kotlin
2.) MVVM architecture 
3.) Using kotlin Coroutines, 
4.) Retrofit
5.) Room Database 
6.) Beautifull UI *
7.) Bookmark Feature *
8.) Animated Splash Screen (Api 33+ require) *
